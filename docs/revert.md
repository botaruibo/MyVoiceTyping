# 原始系统提示词的通用功能：
1. 原始系统提示词是一个用于指导AI助手的提示词，它可以帮助AI助手更好地理解用户的意图和需求。
2. 原始系统提示词可以包含一些预设的指令和规则，例如：
   - 指令：请使用和用户相同的语言回答问题
   - 规则：请不要回答任何与问题无关的内容
3. 原始系统提示词可以包含一些预设的上下文信息，例如：
   - 上下文：你是一个AI助手，你的名字是xxx，你来自xxx，你是xxx
4. 原始系统提示词可能包含一些预设的回答模板也可能不包含：
5. 基于原始系统提示词，对用户的原始输入 raw_input 进行处理后将得到最终的输出 final_output。

# 原始系统提示词的特定功能：
已知的一些原始系统提示词的特定功能：
1. 会将你的话转化为精致的消息、电子邮件和文档，读起来就像你仔细输入的一样
2. 能去掉多余的空格和标点符号，让你的话看起来更简洁、清晰
3. 能去掉多余的单词、短语、句子，让你的话看起来更简洁、清晰，但不丢失重要信息
4. 能去掉重复的单词、短语、句子，让你的话看起来更简洁、清晰，但不丢失重要信息
5. 能直接给出具体的结果，去掉中间的一些临时或错误的表述，给出最终的结果
6. 能编辑修正任何语法或文字的错误，对不合理的句子和词语能更正，进行合理的调整。保持整体语句的通顺流程和符合语法规范。
7. 能编辑优化优化修正语句，使语句符合正常的事实逻辑。对无意义的字词进行合理的猜测，进行更换、去除和增加。错别字词修正不在该范围内。
   - 更换：会绝对保持语句的整体意思不变，但是更换一些无意义的字词，使其符合正常的叙事逻辑
   - 去除：不会去除改变语义逻辑的任何字词。如否定词等
   - 增加：只能补充不改变原含义的字词，如连接、转折等。且只会增加个别字词，不会大量增加。
8. 原始提示词会更具语义和上下文中的{scene}，给出更符合语气的改写。 如在社交/游戏场景下，会更具社交语气；在办公类场景会更正式口吻；其它未指定场景，则不做语气修正
