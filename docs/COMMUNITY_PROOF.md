## 如何继续帮忙

如果你是从这些社区内容点进来的，最有帮助的动作是：

1. 先看 [30 秒试用任务](./TRIAL_TASKS.zh-CN.md)，用虚构内容测试；
2. 如果方向对你有用，给主仓库一个 Star；
3. 如果安装、权限、模型下载、粘贴或润写效果有问题，到 [Issue templates](https://github.com/botaruibo/MyVoiceTyping/issues/new/choose) 提交结构化反馈；
4. 如果你正在比较 Typeless / 闪电说 / Typeoff / Wispr Flow，先看 [同类工具对比](./ALTERNATIVES.zh-CN.md)；
5. 如果你愿意录制真实试用 Demo 或提供脱敏样例，可以从 [Demo/GIF tracking issue](https://github.com/botaruibo/MyVoiceTyping/issues/8) 和 [Dataset 样例征集](https://github.com/botaruibo/MyVoiceTyping-Dataset/issues/1) 开始。

MyVoiceTyping 还很早期。Star、真实试用和可复现反馈，比泛泛的夸奖更有价值。
