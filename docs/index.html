开源 · 本地优先 · 0 费用

MyVoiceTyping：不是泛用版 Typeless 克隆的中文语音输入项目

面向 macOS 的开源中文语音输入工具：按住快捷键说话，松开后完成识别、标点恢复、轻量纠错/润写，并粘贴到当前输入框。它更聚焦中文 / 中英混合、本地润写模型、公开数据集和 Self-Evolution，适合长文本输入、AI Coding prompt、笔记和日常中文写作。

GitHub 项目 / Star 下载 DMG 查看完整介绍

本地数据安全

优先让音频、转写文本和用户确认后的改写留在本机，更适合对隐私、会议纪要、内部 prompt 有顾虑的场景。

中文润写模型

配套开源/公开的中文输入文本润写模型，关注口语到书面语、标点恢复和轻量纠错。

可自进化

alpha-0.03 开始推进本地自进化：用户确认后的修改写入本机 voice_history.jsonl，经用户确认后可参与本机 MLX LoRA 调优，并转换为 GGUF Q4 本地模型。

快速入口

从这里开始：文档导航 快速上手 和 Typeless / 闪电说等方向的差异 ModelScope：MyVoiceTyping-1.5b-q4 训练数据集：MyVoiceTyping-Dataset
