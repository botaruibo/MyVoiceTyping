# Support / 反馈与支持

感谢你试用 MyVoiceTyping。这个项目仍在早期阶段，最需要的是可复现、可公开、已脱敏的真实使用反馈。

请不要在公开 Issue、Discussion、截图或日志里提交公司机密、客户信息、Token、密码、私聊原文、真实内部代码、未授权音频或其他敏感内容。

## 我应该去哪里反馈？

| 你想做什么 | 推荐入口 |
|---|---|
| 先判断是否适合自己 | [30 秒试用任务](docs/TRIAL_TASKS.zh-CN.md) |
| 安装、权限、模型下载、录音、粘贴出问题 | [Bug report / 故障报告](https://github.com/botaruibo/MyVoiceTyping/issues/new/choose) |
| 反馈普通中文输入体验 | [体验反馈 / User feedback](https://github.com/botaruibo/MyVoiceTyping/issues/new/choose) |
| 反馈 AI Coding prompt 场景 | [AI Coding prompt 反馈](https://github.com/botaruibo/MyVoiceTyping/issues/5) |
| 不确定是不是 bug，只想聊使用场景 | [Discussions 体验招募帖](https://github.com/botaruibo/MyVoiceTyping/discussions/2) |
| 提交可公开的 ASR / 润写样例 | [Dataset 样例征集 Issue](https://github.com/botaruibo/MyVoiceTyping-Dataset/issues/1) |
| 反馈安全 / 隐私问题 | 先看 [Security Policy](SECURITY.md) 和 [Privacy / Data Safety](docs/PRIVACY.md) |
| 提功能建议 | [Feature request](https://github.com/botaruibo/MyVoiceTyping/issues/new/choose) |

## 最有价值的反馈格式

```text
使用场景：AI Coding / 工作消息 / 中英混合 / 隐私输入 / 自进化 / 其他
macOS 版本：
Mac 型号 / 芯片：
MyVoiceTyping 版本：

原始口述大意：
实际输出：
我手动改成：

最大问题：
是否愿意继续用：
如果没有 Star，主要原因：
```

## 反馈前的安全检查

- 用虚构或脱敏内容复现问题；
- 不上传真实音频；
- 不贴真实客户、公司、个人或私聊内容；
- 不贴 Token、密码、API key、内部 URL、私有仓库路径；
- 如果必须说明隐私 / 安全风险，只提供最小可复现信息。

## 我只是路过，怎么支持项目？

如果你在找 Typeless / 闪电说之外的开源、本地优先中文语音输入方案，可以：

1. 用 [30 秒试用任务](docs/TRIAL_TASKS.zh-CN.md) 快速判断是否适合；
2. 如果方向对你有用，在 GitHub 点一个 Star；
3. 把可公开、已脱敏的失败样例反馈回来；
4. 如果你写过 Typeless、闪电说、Typeoff、AI Coding prompt 或本地语音输入相关内容，欢迎引用 [Community sharing](docs/COMMUNITY_SHARING.md) 里的克制版介绍。

Star 对早期项目很重要：它会帮助更多 macOS 中文用户发现这个本地优先、0 费用、App / 模型 / 数据集公开的语音输入项目。
